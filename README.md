# Users page app

This is a simple one-page CMS app, created using RTK, RTKQuery and json-server

## Getting started

1. In the `frontend` folder run:
  ```bash
    npm i
  ```
2. In the `api` folder run:
  ```bash
    npm i
  ```
3. In the `frontend` folder run:
  ```bash
    npm start
  ```
4. In the `api` folder run:
  ```bash
    npm start
  ```
5. Happy hacking!
